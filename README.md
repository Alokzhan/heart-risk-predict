# ❤️ Heart Risk Prediction using Machine Learning

## 📌 Overview
This project focuses on predicting the risk of heart disease using Machine Learning techniques. It analyzes various medical parameters such as age, cholesterol, blood pressure, and other health indicators to determine whether a person is at risk.

The goal of this project is to assist in early detection of heart-related diseases and demonstrate the use of ML in healthcare.

---

## 🚀 Features
- Predicts heart disease risk (Binary Classification)
- Data preprocessing and feature analysis
- Model training and evaluation
- Easy-to-understand ML workflow
- Scalable for real-world applications

---

## 🧠 Machine Learning Workflow
1. Data Collection (CSV Dataset)
2. Data Preprocessing  
   - Handling missing values  
   - Feature selection  
3. Train-Test Split  
4. Model Training  
5. Model Evaluation  
6. Prediction  

---

## 🤖 Algorithms Used
- Logistic Regression  
- Decision Tree  
- Random Forest  

---

## 📊 Dataset
The dataset contains medical attributes such as:
- Age  
- Sex  
- Chest Pain Type  
- Cholesterol  
- Blood Pressure  
- Maximum Heart Rate  

**Dataset Source:** UCI / Kaggle Heart Disease Dataset
